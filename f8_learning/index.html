Responsive web (thiết kế web theo các kiểu giao diện)

Cú pháp viết trong css

@media keywords mediatype and (mediafeature and|or|not mediafeature {
CSS Code;
}

1. Keywords

not: Loại Trừ vd có A, B, C not A loại trừ A áp dụng lên B, C
only: Chỉ đây là chế độ mặc định nếu không chỉ thị
and: Và
or: Hoặc

2. mediatype

Print: CSS cho chế độ in
Screen: Css cho màn hình
all - default: Chế độ mặc định tất cả các mediatype
speech: Mà hình hói thường dành cho đối tượng khuết tật

3. mediafeature Được viết trong cặp ()

min-width: kích thước tối thiếu hay còn hiểu là ap dụng cho những màn hình tối thiểu giá trị min-width
max-width: Kích thước tối đa: áp dụng cho những kích thước nhỏ hơn hoặc bằng max-width

VD: @media only screen and (max-width:1024px) {
CSS code;
}
